## Notes on source use policy
- Only datasets with defensible property-level relevance are used in core classification logic.
- Aggregate/contextual datasets are documented and may inform scenario narratives but are not forced into property-level inference.
- Inaccessible/paid sources are documented explicitly with fallback logic (especially OS GB Address).
